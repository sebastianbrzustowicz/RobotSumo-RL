# Contributing to Robot-Sumo-RL

Thank you for your interest in contributing! Please follow these simple steps:

### Contribution Steps
1. **Create a Branch**: Use a descriptive name like `feature/new-reward-logic` or `fix/lidar-noise`.

2. **Commit Changes**: Follow [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) (e.g., `feat: add opponent velocity to state vector`).

3. **Verify**: Run `make lint` and `make test` (make your tests).

4. **Push and Open a PR**: Provide a clear description of what your changes achieve.
